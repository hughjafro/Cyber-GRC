## Methodology

All work in this portfolio is structured around the **NIST Cybersecurity Framework (CSF) 2.0**, using its six core functions as the analytical lens:

- **GOVERN** — Policies, roles, and vendor oversight
- **IDENTIFY** — Asset inventory, threat and vulnerability analysis
- **PROTECT** — Safeguards, access control, encryption
- **DETECT** — Monitoring, logging, anomaly detection
- **RESPOND** — Incident response, breach notification
- **RECOVER** — Business continuity, RTO/RPO, resilience

Risk ratings use a **Likelihood × Impact** matrix (1–5 scale), translated to LOW / MEDIUM / HIGH thresholds aligned with common enterprise risk appetite frameworks.
